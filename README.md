# Mindarc Medical App

This project consists of a mobile frontend and a FastAPI backend.

## Project Structure

- `myapp/`: React Native (Expo) mobile application.
- `backend/`: FastAPI backend server.

## Getting Started

### 1. Frontend (myapp)
```bash
cd myapp
npm install
npx expo start
```

### 2. Backend (backend)
```bash
cd backend
pip install -r requirements.txt
python main.py
```

Refer to the README in each subdirectory for more detailed instructions.
